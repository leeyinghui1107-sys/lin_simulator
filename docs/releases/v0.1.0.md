# 林霖动环数据模拟器 v0.1.0 发布说明

## 概览

`v0.1.0` 是全新公开仓库的初始版本。该版本不继承旧项目 Git 历史，保留 TCP 设备数据模拟核心能力，并内置脱敏后的 `simulator.db` 演示快照。

## 主要能力

- 从本地 SQLite 快照一次性加载端口、命令和响应数据。
- 默认监听 `0.0.0.0`，也支持通过 `-ip` 绑定指定本机 IPv4。
- 支持 TCP 字节流缓冲匹配，兼容拆包、粘包和垃圾字节后的重同步。
- 支持每连接独立响应轮转。
- 支持端口级延迟优先于全局延迟。
- 支持每端口连接数限制、读写超时、指数退避重试和优雅关闭。
- 提供 Windows PowerShell 发布构建脚本，输出 8 个平台主程序和 `simulator.db`。

## 隐私处理

- 公开仓库不包含旧 Git 历史。
- 不包含远程数据库连接信息、账号、密码或密钥。
- `simulator.db` 中 `ports.ip` 已统一为 `127.0.0.1`。

## 验证

发布前应执行：

```powershell
.\scripts\check.ps1
.\build.ps1
```
